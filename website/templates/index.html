Private. Cloud. Storage.

With Syndicate users store data privately in the cloud, using any cloud provider. Underlying providers cannot view user data and Syndicate enables fast downloads.

Features

De-centralized

With Syndicate users are not tied down to any single cloud provider and users can seamlessly move between cloud providers.

Secure

User data is always encypted and users can explicitly control access to their data. Cloud providers have no visibility into user data.

Fast

Syndicate enables faster data access by securely using content distribution networks which boost the speed of your data downloads.

Developer API

Application developers can directly build on top of Syndicate and never worry about storing user data or scaling access to that data.

Getting Started

Syndicate Drive

You can simply signup and start using a free 5GB drive. You can choose to store data on Dropbox, Google Storage, Amazon S3/Glacier, or Box.

Applications using Syndicate

Applications that are being built using Syndicate include:


Secure Email

Storing emails in the cloud is convenient and enables access to email from multiple devices. However, users have to trust cloud providers for not leaking or processing their emails. Syndicate enables private cloud storage for email.

Video Sharing

We're building a simple video sharing application on top of Syndicate. You can think of this as your personal NetFlix for sharing videos with your friends. Syndicate will take care of content delivery and performance.

SearchLab

SearchLab is a testbed for running large-scale search experiments. It enables researchers to quickly deploy search systems by re-using common components and run realistic search experiments using large datasets



About

Syndicate is developed at the Computer Science department of Princeton University. Syndicate started off as an academic research project by Jude Nelson under the supervision of Larry Peterson. Syndicate is currently deployed using PlanetLab and VICCI and contributors include: Muneeb Ali, Wathsala Vithanage, John Whelchel, and Illyoung Choi.
