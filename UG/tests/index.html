About Me

I'm a second-year graduate student at Princeton University.  My research interests include wide-area distributed computing systems, such as content distribution networks, distributed file systems, overlay networks, and cloud computing systems.  I'm originally from Oro Valley, Arizona.

I study with Dr. Larry Peterson.  Right now, I'm developing a specialized distributed filesystem called Syndicate that uses CoBlitz as a back-end for storing and caching blocks of data.

Undergrad Experiences

I received my BS in computer science from the University of Arizona in May 2010, with a minor in Mathematics.  I attended the University of Arizona on a President's Award for Excellence scholarship, awarded in 2006.  I graduated Summa Cum Laude with Honors.

My senior thesis was An Improved Multiprotocol Application Data Transfer Service.  I implemented an intelligent file transfer daemon (IFTD) and a protocol framework for it which allows it to use multiple unmodified file transfer protocol implementations to concurrently fetch pieces of files from one or more remote hosts, record features of each protocol's performance and of the file being transferred, and use machine learning to rank protocol utility for a file with a given set of features.  IFTD currently serves as the file transport framework for Raven, a provisioning service currently deployed on PlanetLab. My undergraduate thesis advisor was Dr. John H. Hartman.
